How to make a 1.19 Minecraft Server!










By: Highpurity

Published: 7/7/2022

Last Edited: 7/7/2022





As most of you are aware, the wild update was released a couple of days ago, and the community’s reaction to it, was disappointed – at best. Now, if you are anything like me, then you’ll want to try the update for yourself before forming an actual opinion, which is what I am going to be helping you do today, by showing you all how to setup your very own 1.19 Minecraft Server on your computer.




Step 1: Download Java





The first thing that you need to do when setting up a 1.19 server is to install Java onto your computer. Press on the button below this text, where you’ll be taken to Java 17's Download Page. From here, you’ll want to scroll down until you see “windows x64 installer”, where you’ll download it by pressing on the link across from it.




Next, you'll want to install java by navigating through the simple installer that you just downloaded. If you see this screen, you have successfully installed Java!






Step 2: Install Paper






Next, it's time to download your Minecraft Server Jarfile. For this tutorial, we will be using Paper, as it's one of the best optimized Minecraft Server Jars for 1.19! Press the button below this text to navigate to paper's website, and download the jarfile closest to the top of your screen!




After you've downloaded this jarfile, create a new folder on your computer, and name it something you'll remember.







Next, move your Paper 1.19 Jarfile into your newly created folder, and then run the file!




Step 3: Configure your Server




After waiting a couple minutes for your server to initially start, you'll see a couple files inside of your Minecraft Server folder. The one we care about is called "eula", and it's a text file. Open this file, and then change Eula=False to Eula=True. Then save your changes, and edit the text file.







Now that you've accepted the EULA, you can now start your Server! Double click your Jarfile again, and your server will now start up! If a GUI like this pulls up on your screen, then your server should have started successfully!
